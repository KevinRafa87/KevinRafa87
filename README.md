# 👋 ¡Hola! Soy Kevin Valenzuela

## 🧑‍🏫 Sobre mí
Soy estudiante en formacioin para la tranformacion digital para la docencia tecnica 1 y docente  apasionado por el desarrollo de software y la educación tecnológica. Me especializo en la creación de soluciones que integran programación y herramientas digitales para mejorar los procesos de enseñanza-aprendizaje. Tengo especial interés en la automatización de tareas educativas y en el uso de inteligencia artificial para personalizar la experiencia del estudiante. Busco constantemente innovar en el aula mediante el uso de tecnología.

## 📂 Proyectos

### 📊 Sistema de Seguimiento Académico Automatizado
Aplicación enfocada en automatizar el seguimiento del progreso estudiantil utilizando hojas de cálculo e integración con flujos automatizados.
- Generación de notificaciones personalizadas según el rendimiento
- Integración con n8n para envío de correos automatizados
- Mejora en la comunicación docente-estudiante
